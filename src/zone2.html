Zone 2 - More Complex Privacy

Zone 2 is compromised of steps that are still very doable yet and protect you more than zone 1. These steps are a little more advances and require more work and detachment from accessibility.

Communication

1. Texting is insecure, use iMessage for iPhone or Signal for Android. Signal is open source and offers end to end encryption which keeps you secure
2. Email services are also insecure. Proton mail is and open source program that offers end to end encryption as well. It is a little annoying to transfer emails, but it is a one time thing and you wont have to do it again

Smartphones

1. iPhone is more secure than android out of the box (without any of the permissions we blindly agree to when setting up a phone)
2. Android is more secure if you are tech savvy and want to make custom security on your device
3. DO NOT use any face ID or fingerprint as a password. The traditional password is the most secure

Storage

1. Always wipe your hard drive every month or so
2. Use C-Cleaner to make sure your hard drives are clean and secure

Router

1. DO NOT use default password on router
2. Hide SSID so you can't see the wifi when you look for it, but you have to input it manually which is more work but a lot more secure

Browser

1. Avoid Google Chrome or any chromium-based browser
2. Use Firefox or any secure browser
3. Use u-block origen
4. Make sure flash and java script are disabled
